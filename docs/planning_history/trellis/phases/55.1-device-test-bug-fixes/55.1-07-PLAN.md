(question-filter.service → embedText → embedding model warm-up), (3) builds the candidate context pack
(`buildCandidateContextPack` / `formatCandidateContextPack` from canonical-knowledge.service), (4)
`chatStream(...)` to the provider, (5) on completion runs `classifyAndAnchorIncremental`.

`timeoutSignal`/`composeSignal` wrap the fetch. Provider handshake (TLS/model warm) happens on first fetch.
From app/src/services/db.service.ts: `IndexedDBBackend.init()` opens the DB (async Promise), and the
in-memory mirror is hydrated from object stores. The first query that awaits hydration pays the cost.
