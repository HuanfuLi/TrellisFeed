# Phase 55.1 Plan 11: GAP-A attempt 2 — visualViewport keyboard-follow for ChatInput Summary

Replaced the rejected eased-decay transform (55.1-05) with a continuous `visualViewport`-driven transform so the Ask-screen input bar follows the keyboard's animated rise frame-by-frame instead of teleporting.

## What Was Built

**Task 1 — reworked `chatinput-offset.ts` to a live-inset helper** (commit `e451b85a`)
- Removed the rejected decay helper (`resolveChatInputSettleDistance`) and its cap constant.
- `resolveChatInputOffset({ innerHeight, viewportHeight, viewportOffsetTop })` now returns the live keyboard inset `max(0, innerHeight - viewportHeight - viewportOffsetTop)`. As the keyboard animates open, the geometry changes across the visualViewport event stream, so the returned inset grows frame-by-frame.
- Clamps non-finite/negative geometry to 0; React-free (unit-tests under `node --test`).
- Rewrote the executing test for the live-inset contract: closed → 0, fully open → full inset, mid-animation → strictly-between partial value, NaN/negative → 0, offsetTop subtraction, idempotent, no-react guard, decay-removed guard. **8 passing assertions.**

**Task 2 — continuous visualViewport-driven transform in `ChatInput.tsx`** (commit `d27e3067`)
- Replaced the 55.1-05 transient transform + 0.18s CSS transition with an effect that subscribes to `window.visualViewport` `resize` AND `scroll` events.
- On each event (rAF-coalesced), computes the live inset via `resolveChatInputOffset` and the layout reflow via `document.documentElement.clientHeight`, then sets `translateY(-residual)` where `residual = max(0, liveVisualInset - layoutReflow)`.
- No CSS transition on the transform (motion comes from the event stream). `willChange: transform` retained for compositing.
- Listener cleaned up on unmount; `transform`/`willChange` confined to the `<form>` wrapper (Header sibling).
- Companion lever: added `interactive-widget=resizes-content` to the `index.html` viewport meta so the layout viewport resizes predictably with the keyboard.
