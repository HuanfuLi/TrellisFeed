# Archived: Trellis product-era GSD planning state

This directory is the frozen `.planning/` tree of the **Trellis** product (formerly EchoLearn) as of the fork conversion to **QuestionTrace** on 2026-07-09. It ends at milestone v1.7, Phase 56 shipped / Phase 57 (Rewards) unstarted.

It is **not live GSD state** — the live state for the QuestionTrace research prototype lives in `.planning/` at the repo root. Nothing here governs current work; the features these phases describe (podcast, flashcards/SRS, planner/trellis gamification, rewards shop, global chat, collections, live news/YouTube) were pruned per `docs/research_system_design.md` §15.3 and must not be resurrected.

Kept for reference: phase history, verification reports, retrospectives, and requirement traceability of the ancestor product. The durable operational knowledge (load-bearing invariants, lessons) was carried into the root `CLAUDE.md` and `docs/prune_report.md` during Phase 0 of the conversion.
