# Stack

## Core Technologies
- **Language**: [TypeScript](https://www.typescriptlang.org/) (v5.9.3)
- **Framework**: [React](https://react.dev/) (v19.2.0)
- **Runtime**: [Node.js](https://nodejs.org/) (for dev/tests)
- **Build Tool**: [Vite](https://vitejs.dev/) (v7.3.1)
- **Native Bridge**: [Capacitor](https://capacitorjs.com/) (v8.1.0)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (v4.2.1)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) (v12.38.0)
- **Routing**: [React Router](https://reactrouter.com/) (v7.13.1)

## Data & Storage
- **Native Database**: [SQLite](https://github.com/capacitor-community/sqlite) (@capacitor-community/sqlite v8.0.1)
- **Web Fallback**: `localStorage` (simulated key-value store)
- **Knowledge Mapping**: [Mind Elixir](https://mind-elixir.com/) (v5.9.3)

## AI & NLP
- **Markdown Rendering**: [React Markdown](https://github.com/remarkjs/react-markdown) (v10.1.0)
- **Markdown Plugins**: [Remark GFM](https://github.com/remarkjs/remark-gfm) (v4.0.1)
- **Providers**: Claude, Gemini, OpenAI, Local, LM Studio

## Tooling
- **Linting**: [ESLint](https://eslint.org/) (v9.39.1, Flat Config)
- **Types**: [TypeScript ESLint](https://typescript-eslint.io/) (v8.48.0)
- **Testing**: [Node.js Test Runner](https://nodejs.org/api/test.html) (built-in)
