    **3. Create app/src/services/web-search.service.ts:**

    - Import `Capacitor` and `CapacitorHttp` from `@capacitor/core` (same pattern as `localPost` in llm/index.ts)
    - Import `settingsService` from settings.service.ts
    - Import `ServiceResult, WebSearchResult, WebSearchResponse, SourceCitation` from types

    Implement `webSearch(query, options?)`:
    - `options` type: `{ topic?: 'general' | 'news'; maxResults?: number }`
    - Read API key from `settingsService.getSync().webSearch?.tavilyApiKey`
    - If no key, return `{ success: false, error: { code: 'NOT_CONFIGURED', message: 'Tavily API key not set. Add it in Settings > Web Search.', retryable: false } }`
    - POST to `https://api.tavily.com/search` with body: `{ query, topic: options?.topic ?? 'general', max_results: options?.maxResults ?? 5, search_depth: 'basic', include_answer: false, include_raw_content: false }`
    - Headers: `{ 'Content-Type': 'application/json', 'Authorization': 'Bearer ${apiKey}' }`
    - Use CapacitorHttp.post on native, fetch on web (same branching as localPost in llm/index.ts)
    - Map response to `WebSearchResponse`
    - Wrap in try/catch, return `ServiceResult<WebSearchResponse>`
    - Add `serviceName: 'web-search'` as a comment for future token tracking

    Implement `extractCitations(content: string): { body: string; sources: SourceCitation[] }`:
    - Look for `\n\s*(?:Sources|References)\s*:\s*\n` pattern in content
    - If not found, return `{ body: content, sources: [] }`
    - Split content at the divider — everything before is body, everything after is source lines
    - Parse each source line with regex: `/\[(\d+)\]\s*\[?([^\]\n]+?)\]?\s*(?:\(([^)\n]+)\)|[-:]\s*(https?:\/\/\S+))/gm`
    - Return `{ body: body.trimEnd(), sources }`

    Export: `webSearch`, `extractCitations`
