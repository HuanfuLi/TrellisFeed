## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| translator → locale bundle | Translation output is added to JSON files committed to the repo. Trust = developer-reviewed Sonnet subagent output. |
| user input → search query | Search query is passed to Fuse.js (in-memory). No LLM, no network, no eval. Fuse handles arbitrary input safely BUT very long strings (paste-attack) can hang the WebView. |
| user input → collection name | (Validated in 50-03; tested here as a scaffold.) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-50-XSS-NAME | Tampering / XSS | Collection name rendered in picker / drill-in / toast | mitigate | Validated in 50-03 service; tested at picker render boundary via `CollectionPickerSheet.test.mjs` source-reading assertion (no `dangerouslySetInnerHTML`). |
| T-50-XSS-HL | Tampering / XSS | Highlighted match indices spliced into JSX | mitigate | Tested at `HighlightedText.test.mjs` source-reading assertion (no `dangerouslySetInnerHTML`; matched runs wrap text in `<mark>` JSX nodes — React escapes by default). |
| T-50-QUERY-DOS | Denial of Service | Fuse.js search query length | mitigate | `library-search.service.test.mjs` asserts a 200-char cap is enforced before query is passed to Fuse. |
</threat_model>
