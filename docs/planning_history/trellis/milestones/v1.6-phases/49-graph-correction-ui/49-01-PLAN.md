    Factory state machine logic (shared between hook + factory consumers):

    `onPointerDown(e)`:
    - Reset `didLongPress` and `didDrag` to false; record `startCoord = { x: e.clientX, y: e.clientY }`; record `lastCoord = startCoord`.
    - Schedule `setTimeout(() => { didLongPress = true; void hapticImpactLight(); }, longPressMs)`.

    `onPointerMove(e)`:
    - Update `lastCoord = { x: e.clientX, y: e.clientY }`.
    - If `startCoord == null`, return (defensive — no pointerdown).
    - Compute `dx = e.clientX - startCoord.x`, `dy = e.clientY - startCoord.y`, `dist = Math.hypot(dx, dy)`.

      - Else if `didLongPress && !didDrag` (post-480ms, first move): set `didDrag = true`; invoke `onDragStart(startCoord.x, startCoord.y)`.
      - If `didDrag`: invoke `onDragMove(e.clientX, e.clientY)`.

    `onPointerUp(e)`:
    - Clear timer.
    - If `didLongPress && didDrag`: invoke `onDragEnd(e.clientX, e.clientY)`.
    - Else if `didLongPress && !didDrag`: invoke `onLongPressRelease(e.clientX, e.clientY)`.
    - Else: tap path — do nothing here (let the existing click handler at GraphScreen.tsx:302 handle it).
