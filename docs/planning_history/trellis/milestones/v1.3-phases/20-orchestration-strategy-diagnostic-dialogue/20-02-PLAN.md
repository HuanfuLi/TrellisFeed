       **Exported service object `diagnosticDialogueService`:**

       - `startSession(initialText: string): Promise<DiagnosticSession>` —
         1. Extract signals from initialText via extractSignals
         2. Create session with id = `diag-${Date.now()}`, status 'active'
         3. Add initial user turn with extracted signals
         4. Persist to localStorage
         5. Return session

       - `generateFollowUp(session: DiagnosticSession): Promise<string>` —
         1. Build prompt with system message: learning coach persona, include confusion/curiosity from mergedSignals, include the user's last message
         2. Call `chatCompletion` with serviceName `'diagnostic'`
         3. Return the follow-up question string
         4. Add assistant turn to session, persist

       - `processReply(session: DiagnosticSession, reply: string): Promise<DiagnosticSession>` —
         1. Extract signals from reply
         2. Merge with existing session.mergedSignals
         3. Add user turn with new signals
         4. Count user turns; if >= MAX_TURNS, set status to 'completed'
         5. Persist updated session
         6. Return updated session

       - `finalize(session: DiagnosticSession): DiagnosticSession` —
         1. Set status to 'completed'
         2. Clear from localStorage
         3. Return finalized session

       - `getActiveSession(): DiagnosticSession | null` — load from localStorage

       - `clearSession(): void` — remove from localStorage
