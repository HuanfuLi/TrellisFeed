# Trellis — 5分钟功能演示视频脚本与录制流程指南

**目标受众：** 普通用户、授课教授、投资者。
**视频定位：** 讲解式功能演示，风格口语化（Informal），直接聚焦 App 具体交互功能，避开多余修饰。
**视频时长：** 约 5 分钟。

---

## 📋 第一部分：实机演示画面操作指引 (VISUAL)

按照以下操作顺序录制屏幕，每个动作应预留 1–2 秒停顿，避免画面闪烁太快：

### 1. 首页推荐流交互 (0:00 - 0:35)
1. **画面**：打开 App，直接呈现在 `/home`（首页）页面。
2. **操作**：慢速下滑首页，展示瀑布流（Masonry Feed）中的各种卡片（包含 YouTube 视频卡片、新闻卡片与纯文本卡片）。
3. **操作**：双击其中一张卡片（演示“点赞/喜欢”反馈效果）。
4. **操作**：长按另一张卡片弹出菜单，点击“保存”，在弹出的收藏夹分组表单 (`CollectionPickerSheet`) 中选择一个分类，将其归档。

### 2. 现场提问与智能解析 (0:35 - 1:20)
1. **画面**：点击底部栏“提问”标签，进入 `/ask` 页面。
2. **操作**：展示输入框上方的快捷问题模板卡片。
3. **操作**：长按底部麦克风图标，现场说出问题：**“费曼学习法在日常自主学习中怎么落地？”**。松手，看到语音正确转写为文本并发送。
4. **操作**：等待 AI 流式回复生成。

### 3. 可视化图谱与 Learn as Post (1:20 - 2:15)
1. **画面**：点击底部“图谱”标签，进入 `/graph` 页面，展示 MindElixir 导图。
2. **操作**：点击节点展开/折叠按钮展示交互。
3. **操作**：长按叶子节点（刚刚提问的“费曼学习法”），将其拖拽到另一个大类节点下（显示高亮磁吸效果），松手确认重组。
4. **操作**：点击底部悬浮的“撤销” (Undo) 按钮，看到节点流畅地飞回原位。
5. **操作**：点击刚才拖动的“费曼学习法”概念节点，底部划出卡片面板。点击面板，页面切换进入该概念的详情页 `/anchor/:id`。
6. **操作**：在详情页中，点击 **“作为帖子学习” (Learn as Post)** 按钮。页面跳转至 `/posts/:id`，向下滑动，展示 AI 整合该概念下所有历史问答生成的系统化阅读指南。

### 4. 规划器与植物状态系统 (2:15 - 3:00)
1. **画面**：点击导航栏进入“规划器” `/planner` 页面。
2. **操作**：指向顶部通过 SVG/Canvas 渲染的虚拟知识树 `TrellisHero`。
3. **操作**：展示右侧数据面板（当前金币/樱币 credits、概念总数和已遗忘概念数）。
4. **操作**：下滑展示“建议动作”列表。点击其中一个已遗忘（Dead）概念右侧的“修剪” (Prune) 按钮，看到弹窗或 Toast 确认，该叶子在树上归档消失。

### 5. 间隔重复复习与闪卡库 (3:00 - 3:40)
1. **画面**：在建议动作列表点击一个“治愈” (Heal) 按钮，或者从首页入口切入 `/review` 复习页面。
2. **操作**：展示正面卡片问题，点击卡片使其 3D 翻转露出背面答案。
3. **操作**：点击底部四个评分按钮之一（如 Hard 或 Good），下一张卡片滑入。答完后看到 Confetti 烟花特效爆发。
4. **操作**：切到顶部的“闪卡库” (Library) 标签页，滑动闪卡列表。点击某张闪卡的“大头针”图标进行置顶 (Pin)，再点击删除图标进行删除演示。

### 6. 今日播客与睡前自动生成 (3:40 - 4:30)
1. **画面**：进入 `/podcast` 播客页面。
2. **操作**：点击播放按钮（此时应能听到播客对话的音频传出），展开配置面板展示长度（标准/深度/延伸）与风格（聚焦/对话/复习）配置项。
3. **操作**：展开“查看文稿”面板，下滑滚动展示 AI 生成的双人剧本和文字逐字对照。
4. **操作**：随后切换到 `/settings`，然后点击进入“功能设置” (`/settings/features`），展示 **“就寝时间” (Sleep Time)** 选项并作短暂停留。

### 7. 模糊检索与大模型配置 (4:30 - 5:00)
1. **画面**：导航到 `/saved`（收藏与搜索）页面。
2. **操作**：展示 Saved、History（按天分组）与 Collections 标签。在顶部搜索框中打字搜索，点击筛选芯片（如 Concept 或 Date）展示 Fuse.js 模糊匹配高亮效果。
3. **操作**：导航到 `/settings/ai`，展示云端 API 密钥框与本地大模型端点（Ollama / LM Studio）的配置输入框。
4. **操作**：重新切回 `/graph` 页面，缓慢缩小并渐隐结束。

## 🎤 第二部分：配套完整录音旁白 (VOICE OVER)

*提示：录音时请保持语气自然、放松，像是在和朋友介绍自己写的小工具。*

### 【第一段：首页推荐流】 (0:00 - 0:35)
“哈啰大家！今天带大家实机体验一下 Trellis。这是一个本地优先、隐私安全的智能学习助手。

进入首页后，我们首先看到的是推荐卡片流。系统会根据我们最近关心的概念，自动从 YouTube 视频、实时新闻和网页里整理出卡片。如果看到感兴趣的内容，可以直接双击卡片进行点赞，或者长按弹出菜单，把它一键保存到我们自定义的分组收藏夹里，整理起来非常直观。”

### 【第二段：现场提问与卡片提取】 (0:35 - 1:20)
“如果我们想开始学习一个新概念，可以切到提问页面。除了这些预设的模板，我们也可以长按底部的麦克风直接语音提问：*‘费曼学习法在日常自主学习中怎么落地？’*

发送之后，AI 会立即整理出答案。在回答的同时，系统已经在后台自动为我们生成了复习闪卡，并且提取了新概念更新到我们的首页卡片流里，免去了手动复制粘贴的繁琐。”

### 【第三段：知识图谱与 Learn as Post 整合】 (1:20 - 2:15)
“现在我们进到图谱页面。刚才提问的所有问题，已经被自动整理成了思维导图的分支，并且按逻辑关系进行了聚类。

这是一个完全可编辑的画布。比如我们可以长按刚刚的‘费曼学习法’节点，随意把它拖拽到其他分支上进行重组；如果不小心拖错了，点一下底部的‘撤销’按钮它就会飞回原位。

不仅如此，点击‘费曼学习法’节点并进入它的概念详情页，这里有一个‘作为帖子学习’的按钮。点击它，系统就会把这概念下的所有零散问答一键聚合成一篇完整的系统化指南。这对于串联知识碎片、做整理复习非常有用。”

### 【第四段：规划器与植物状态系统】 (2:15 - 3:00)
“再来看看规划器。顶部的这棵虚拟藤架代表了我们目前的记忆状态：每个叶子代表一个知识点，绿色代表记忆牢固，黄色代表开始衰退，灰色则代表已经遗忘。

对于已经不再关注的失效概念，我们可以在下方建议动作里直接点击‘修剪’。这样该节点就会从树上归档消失，首页也不会再向我们推荐相关的卡片，让我们保持高度专注。”

### 【第五段：间隔重复复习】 (3:00 - 3:40)
“对于黄色或灰色的节点，我们可以点击‘治愈’直接开始闪卡复习。卡片正面是提问衍生出来的问题，点击一下就会翻转露出答案。

复习时我们根据掌握程度打分，系统会自动推算下一次复习时间。复习完所有卡片，就会有一个精美的烟花庆祝动效。在旁边的‘闪卡库’中，我们还能快速搜索、置顶重要卡片或删除不需要的内容。”

### 【第六段：今日播客与睡前自动生成】 (3:40 - 4:30)
“播客功能是 Trellis 的一大亮点。系统能把今天新学的知识点，一键生成为对话式播客。

我们可以直接点击播放并展开面板。除了手动调整播客的长度和风格之外，我们还可以在设置中设定就寝时间，这样系统就会在睡前自动为我们提前生成当天的复习播客，盲听非常方便。我们还可以一边听，一边下滑查看 AI 生成的稿子。”

### 【第七段：检索与大模型配置】 (4:30 - 5:00)
“最后，在收藏和历史记录里，我们可以通过模糊搜索和概念标签，高亮快速定位过往内容。

在设置里，除了主流的云端 AI，Trellis 还支持 Ollama 和 LM Studio 等本地运行的大模型端点。所有的思考和学习记录都会以 SQLite 格式存在本地设备上，绝不泄露隐私。

Trellis 帮你留住每一次好奇心，谢谢大家！”

## 📋 第三部分：演示数据准备材料清单 (Checklist)

为确保演示过程中，每一个功能操作与系统反馈都完美契合解说词，请提前准备好以下数据：

### 1. 预录入历史问答数据（构建基础图谱）
在正式录制前，前往 `/ask` 页面，逐个输入并发送以下问答。这将帮助你在 `/graph` 中生成分类良好的主题分支，并在演示现场提问时提供正确的引用文献关联：

*   **费曼学习法与核心步骤（概念主枝）**
    1.  问题：“费曼学习法核心的四个步骤是什么？”
        *答：系统将在图谱中建立‘费曼学习法’概念节点。*
    2.  问题：“在向他人或 AI 解释概念时，如何快速发现自己的知识盲区？”
        *答：在图谱中生成相关的子关联节点。*
*   **记忆与认知学习主题（知识分类）**
    3.  问题：“什么是艾宾浩斯遗忘曲线？它如何指导我们的复习计划？”
    4.  问题：“为什么说主动召回（Active Recall）比被动阅读更有效？”
        *答：在认知科学分支中生成聚类节点，使图谱树形结构丰满。*

### 2. 现场提问动作准备
*   **现场输入问题**：“费曼学习法在日常自主学习中怎么落地？”
*   **配置验证**：预录入前两个关于费曼学习法的问题后，在录屏现场用语音发送该问题，能正确触发已有节点的关联。

### 3. “作为帖子学习”功能验证 (Learn as Post)
*   **操作**：在 `/graph` 页面中点击“费曼学习法”节点，底部会弹出摘要详情卡片。
*   **验证**：点击底部详情面板成功导航到 `/anchor/:id` 页面，点击“作为帖子学习”按钮，确认能够正常检索数据库，并将前述预录入的问答（例如“费曼学习法核心的四个步骤”）聚合成 Markdown 帖子。

### 4. 今日播客预生成 (AI Podcast)
*   **操作**：录制前进入 `/podcast` 页面，生成一期播客。
*   **校验点**：勾选“费曼学习法”和“遗忘曲线”概念，使播客音频讨论的内容与前文呼应。生成完毕后，暂停并拉回到 0:00 以备现场播放。

### 5. 睡前自动生成播客配置
*   **步骤**：录屏展示 `/settings/features`，在“何时生成每日播客”中预设就寝时间（例如 `22:00`），配合讲解。

### 6. 闪卡复习队列预热 (SRS Queue)
*   **操作**：在 `/review` 的“闪卡库”中确认“费曼学习法”相关的卡片已生成。
*   **调整**：如果卡片未处于“Due”状态，可将系统时间往后调整 1 天，使得卡片在规划器和首页显示为待复习状态。

### 7. 设备与模型配置
*   **主题**：将系统切换为 Dark Mode（深色模式）。
*   **模型**：将提问模型设为 OpenAI `gpt-4o-mini` 或 Gemini `1.5-flash`，以保证语音转写和流式回复顺畅极速。
