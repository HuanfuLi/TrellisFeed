Extend the offline content pipeline so the frozen pool carries the typed §10.4 global content graph and per-post ranking features (D-01). The app then imports a ready-made global graph with zero in-app graph/embedding compute.

The current exporter emits only untyped post_concept_edges.json / post_claim_edges.json helper files, which cannot distinguish explains vs mentions or supports vs challenges. Because a frozen pool is immutable, these artifacts must exist in the exporter BEFORE the first approved freeze (Phase 2 plan 02-09 performs the actual freeze).

Output: schemas + compiler + freeze/verify integration + pipeline tests.
</objective>
