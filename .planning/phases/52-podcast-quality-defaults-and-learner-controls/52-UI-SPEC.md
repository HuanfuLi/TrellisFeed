## Color

Values mapped from existing custom theme (`index.css`):

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FFFBF5` / `#0A0A0A` | Background, app surfaces |
| Secondary (30%) | `#FFFFFF` / `#181818` | Cards, inner containers |
| Accent (10%) | `#558B2F` / `#4CAF50` | Primary buttons, active chip selections, selected tabs |
| Destructive | `#E53935` / `#EF5350` | Destructive actions only (e.g. Delete) |

Accent reserved for: Primary interactive surfaces, selected chip background, primary CTA. Never for all interactive elements (unselected chips use secondary/muted colors).
