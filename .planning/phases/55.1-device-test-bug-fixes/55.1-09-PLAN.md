animations too, and wire real visibility into the hardcoded `inView` so the perf guard engages.

and a mandatory on-device UAT (toggle Trellis Dev Mode ON, confirm smooth cross-screen swipe + feed
scroll). Do NOT re-touch the blossom-date build path — 55.1-06 already did that correctly.
