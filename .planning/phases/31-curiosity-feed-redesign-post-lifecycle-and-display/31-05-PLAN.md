    **SVG Vine Illustration:**

    The vine is an inline SVG element. Key dimensions per UI-SPEC:
    - Inline mode: 48px height
    - Compact mode: 36px height
    - Potted plant: left anchor, 24x24px. Draw a simple potted plant: rectangle pot + triangular leaves above
    - Vine stem: horizontal line from plant rightward, 3px stroke, `var(--primary-40)`. Width = `(explored / total) * availableWidth`
    - Leaves: small leaf shapes (8x12px path elements) at even intervals along the stem. Fill `var(--primary-40)` at 60% opacity
    - Flowers: 10x10px circles at concept milestone positions. Fill `var(--primary-40)` when that concept is explored, fill `var(--muted)` when unexplored
    - Completion state (D-05): when `isComplete`, vine stroke color changes to `#E8A838`, all flowers get a CSS pulsing scale animation (1.0 to 1.3, 600ms ease-in-out, infinite), small fruit icons (12x12px circles with `#E8A838` fill) appear

    **Container styling per UI-SPEC:**
    - Inline mode: `background: 'var(--card)'`, `borderRadius: 'var(--radius)'`, `boxShadow: 'var(--shadow-1)'`, `padding: '16px'`
    - Compact mode: `background: 'var(--card)'`, `boxShadow: 'var(--shadow-1)'`, `padding: '12px 16px'`

    **Expand/collapse interaction (D-03):**
    - `useState` for `expanded` boolean
    - Tap vine area → toggle `expanded`
    - ChevronDown icon (16px, `var(--muted-foreground)`) rotates 180deg when expanded via `transform: rotate(${expanded ? 180 : 0}deg)` with `transition: 'transform 200ms ease-out'`
    - Click-away listener: `useEffect` that adds `document.addEventListener('click', handleClickOutside)` and cleans up on unmount. Use a ref on the container to check if click was inside.

    **Concept checklist (expanded state per D-03, D-04):**
    - Renders below the vine SVG, 8px top margin
    - Wrapped in a div with `maxHeight: expanded ? '216px' : '0'`, `overflow: 'hidden'`, `transition: 'max-height 200ms ease-out'`
    - Role: `role="list"`, each item `role="listitem"`
    - Each item: 36px height, concept name (14px weight 400)
    - Explored items: `textDecoration: 'line-through'`, `color: 'var(--muted-foreground)'`, `aria-disabled="true"`
    - Unexplored items: `color: 'var(--foreground)'`, tappable → calls `onConceptTap(concept.id)` which scrolls to first post for that concept (D-04)
    - Max 6 visible items (216px / 36px = 6), overflow-y auto for scrolling

    **History icon (D-37 entry point):**
    - In inline mode only: `Clock` icon (20px, `var(--muted-foreground)`) positioned top-right of the card
    - `aria-label` from `t('home.history.iconLabel')`

    - `aria-label` from `t('home.feed.vineProgress', { explored, total })` or `t('home.feed.vineComplete')` when complete
